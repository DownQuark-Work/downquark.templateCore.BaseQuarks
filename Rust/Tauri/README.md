# Development Qonsole

> A dq build

```
Template created! To get started run:
  cd basequark-rust-tauri
  pnpm install
  pnpm tauri android init
  pnpm tauri ios init

For Desktop development, run:
  pnpm tauri dev

For Android development, run:
  pnpm tauri android dev

For iOS development, run:
  pnpm tauri ios dev

```
